# ML Engine – Proyecto VPN + ML

Este módulo contiene los scripts para el **entrenamiento del modelo de detección de anomalías** basado en características del datagrama IPv4.

Incluye:
- Preprocesamiento
- Normalización
- Conversión de IP → entero
- Entrenamiento Random Forest
- Entrenamiento Autoencoder
- Exportación de modelos
